## Installation

Download and compile base_convert.cpp on your respective platform.

### linux
```bash
g++ base_convert.cpp -o base_conversion.out
./base_conversion.out
```
### Windows
```cmd
g++ base_convert.cpp -o base_conversion.exe
base_conversion.exe
```
